# 🛒 NovaCart – Modern React E-commerce Website

NovaCart is a modern and responsive e-commerce web application built with **React**. It provides a smooth shopping experience with features like product browsing, search, filtering, wishlist, cart management, checkout, and more.

---

## ✨ Features

- 🏠 Modern Home Page
- 📦 Product Listing
- 🔍 Product Search
- 🗂️ Category Filtering
- ↕️ Product Sorting
- 📄 Product Details Page
- ❤️ Wishlist Management
- 🛒 Shopping Cart
- ➕ Increase & Decrease Quantity
- 💳 Checkout Page
- ✅ Order Success Page
- ⚡ Skeleton Loading
- 🔔 Toast Notifications
- 💾 Local Storage Support
- 📱 Responsive Design
- 📌 Sticky Navigation Bar

---

## 🛠️ Tech Stack

- React
- React Router DOM
- Context API
- CSS Modules
- JavaScript (ES6+)
- DummyJSON API
- React Icons
- React Hot Toast

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/novacart-react.git
```

Navigate to the project

```bash
cd novacart-react
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
